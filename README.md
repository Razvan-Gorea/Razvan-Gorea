### 🤖 Multi-Agent LLM Application for Unified Data Retrieval
A full-stack AI-powered application that orchestrates LLM agents to ingest and retrieve data from multiple datastores.
- Built with **React**, **FastAPI**, **SQLite**, and **LangGraph**

### 🐚 Custom Unix Shell
A feature-complete Unix shell written in C that emulates core bash functionality.
- Supports built-in commands (`cd`, `echo`, `pwd`) and background process execution via `fork()`
- Interactive and batch mode support with custom tokenization
- Includes a Makefile and comprehensive user manual

---

### 🔍 AI Image Detection Pipeline
A multi-phase ML pipeline that classifies images as **Real** or **AI-Generated**, combining hand-crafted signal-processing features with deep CNN embeddings.
- Fuses 81 handcrafted features (FFT, LBP, DCT, noise residual) with **EfficientNet-B3** embeddings via an MLP fusion head
- Trained on 4 AI-generator datasets: **DALL-E 3, GLIDE, Imagen, Stable Diffusion**
- XGBoost classifier + PCA dimensionality reduction + StandardScaler pipeline
- Evaluated against the cross-generator **Nano Banana** HuggingFace benchmark

---

### ♻️ Waste Type Classification
A deep learning system for classifying waste images into 6 categories, with systematic evaluation of preprocessing and data balancing strategies.
- Trained and compared **ResNet-50, MobileNet V3, EfficientNet-B0**, and a custom **AttentionCNN** with Squeeze-and-Excitation blocks
- Evaluated preprocessing pipelines: Canny edge detection, Sauvola thresholding, and AI-based background removal (`rembg`)
- Addressed class imbalance with **Weighted Sampling, SMOTE, and DeepSMOTE** (latent-space interpolation)
- Model interpretability via **SHAP** value analysis

---

### 📈 Bitcoin Price Forecasting
A time-series ML project forecasting Bitcoin closing prices using classical and ensemble models with lag-based feature engineering.
- Compared **SARIMAX, XGBoost, and Random Forest** across lag windows of 2, 7, 30, and 90 days
- Applied log transformation to stabilise variance across Bitcoin's wide price range (~$68 to ~$63,500)
- Implemented a **rolling walk-forward forecast** to simulate real-world deployment with continuously arriving data
- Evaluated with MAE & RMSE on an 80/20 chronological train/test split
